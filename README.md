# 🔐 Password Strength Checker

A modern web-based **Password Strength Checker** that helps users create secure passwords by analyzing password complexity, entropy, and estimated crack time.

This project demonstrates password security concepts used in cybersecurity and secure authentication systems.

---

## 🚀 Features

✔ Real-time password strength detection  
✔ Password entropy calculation  
✔ Estimated time to crack the password  
✔ Password generator for strong passwords  
✔ Security tips for safe password practices  
✔ Interactive UI with animations  
✔ Responsive design (works on desktop and mobile)  
✔ Dark mode support

## 🔑 Password Security Checks

The application evaluates passwords based on:

- Minimum length
- Uppercase characters
- Lowercase characters
- Numbers
- Special characters
- Password entropy

---

## 💡 Security Tip

A strong password should:

- Be at least **12 characters long**
- Include **letters, numbers, and symbols**
- Avoid dictionary words
- Use **unique passwords for each account**
